# Code Availability Statement

The datasets used in this study are publicly available, and their sources, versions, and annotation formats are described in the manuscript and in this repository.

This repository provides dataset preparation instructions, preprocessing scripts, annotation conversion utilities, dataset-statistics scripts, public training-setting documentation, and module-level pseudocode for CAFU, LTA, and FSCM.

The project is still under follow-up research and development. The public materials focus on documenting the dataset processing workflow, experimental setting, and module-level algorithmic design used in the manuscript.
